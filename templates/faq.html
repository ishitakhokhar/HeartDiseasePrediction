Frequently Asked Questions

Find answers to common questions about HeartAI

General Questions

HeartAI is an AI-powered system that predicts heart disease risk using machine learning. It analyzes health parameters like age, blood pressure, cholesterol, and lifestyle factors to provide risk assessments.

HeartAI is designed for:

  • Individuals for personal health awareness
  • Healthcare professionals as a supplementary tool
  • Researchers and students studying ML in healthcare

Yes! HeartAI is completely free and open-source.

Technical Questions

HeartAI primarily uses:

  • Random Forest (74% accuracy)
  • Logistic Regression (for interpretability)

Feature engineering involves preprocessing raw health data before model training. This includes:

  • Scaling numerical features using StandardScaler
  • Encoding categorical variables
  • Handling missing values
  • Removing outliers using statistical methods

No personal data is stored or shared.

All data is processed in real-time and immediately discarded.
